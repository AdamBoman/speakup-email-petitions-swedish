speakup-email-petitions-swedish
===============================

Swedish localization for the WordPress plugin SpeakUp! Email Petitions by [Kreg Wallace](https://github.com/kregwallace)
[https://github.com/kregwallace/speakup-email-petitions/](https://github.com/kregwallace/speakup-email-petitions/)

Intended for SpeakUp! Email Petitions version **2.4.2**.

Created using [Poedit](http://sourceforge.net/projects/poedit/).

Svenska
-------

Språkfiler på svenska för WordPress pluginen SpeakUp! Email Petitions.
Översättningen är ganska snabbt gjord så det finns säkert utrymme för förbättringar.
